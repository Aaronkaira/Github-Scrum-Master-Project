# Scrum Master Portfolio Project – Agile Simulation

This project simulates managing a Common Website development using Scrum methodology.


It demonstrates the Scrum Master’s skills in:
- Product backlog management
- Sprint planning and execution
- Monitoring metrics (burndown, velocity)
- Facilitating retrospectives
- Team capacity planning
- Reporting


## Tools Used
- Monday.com/Jira / Trello (for backlog simulation)
- Monday.com/ Excel / Google Sheets (for burndown & metrics)
- Markdown documentation for retrospectives and reports
